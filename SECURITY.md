# Security Policy

## Supported Versions

The following versions are currently supported and receive updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0   | :x:                |

## Reporting a Vulnerability

To report a vulnerability create an issue and if you have a solution also create a pull request with the fix.
